# Integração com Mercado Livre

Desenvolvimento de frontend e backend para realização de integração com o Mercado Livre.

## Tecnologias usadas

- React Js
- Redux
- Styled Components
- Axios
- NodeJs
- Express

## Frontend

O frontend se encontra em /front. 

### Rodando o projeto

- npm install
- npm start

## Backend

O backend se encontra em /backend. 

### Rodando o projeto

- npm install
- npm start
